================================================================================
WORKAROUNDS (What to use instead of webhooks)
================================================================================

Option 1: Time-based polling
  - Agent checks external API every N minutes
  - Uses Pi.Scheduler
  - Example: Check stock prices every 15 minutes

Option 2: File-based integration
  - External system writes files
  - Agent monitors folder and processes files
  - Uses FileSystemWatcher
  - Example: Process CSVs dropped in folder

Option 3: Channel integration
  - Use messaging platform (Slack, Discord, etc.)
  - External system posts to channel
  - Agent responds via Pi.Gateway
  - Example: Slack bot receives notifications
