## Deployment Instructions

### Prerequisites
- .NET 8.0 SDK
- Windows 10+ / Linux / macOS 10.14+

### Build
```bash
cd /path/to/pi-mono-net
dotnet build
```

### Run
```bash
cd src/Pi.CodingAgent
dotnet run
```

### Usage
```bash
# Start the coding agent
dotnet run --project src/Pi.CodingAgent

# The agent will:
# 1. Prompt for GitHub Copilot authentication (first time)
# 2. Start interactive REPL
# 3. Accept commands and natural language
# 4. Auto-save sessions to ~/.pi/sessions/
# 5. Store settings in ~/.pi/settings.json
# 6. Encrypt credentials in ~/.pi/auth/
```
