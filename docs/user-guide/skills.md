# Skills Guide
Working with skills and SKILL.md files.

## What are Skills?
Skills are reusable knowledge modules that extend agent capabilities.

## Creating a Skill
```markdown
---
name: my-skill
description: Brief description
---

# My Skill
Detailed instructions...
```

## Skill Locations
1. Project skills: `.agents/skills/`
2. User skills: `~/.agents/skills/`

See [Creating Skills](../technical/creating-skills.md) for technical details.
