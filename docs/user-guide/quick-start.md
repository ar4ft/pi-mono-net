# Quick Start Tutorial

Get your AI coding agent up and running in 5 minutes!

## Step 1: Start the Agent

```bash
cd src/Pi.CodingAgent
dotnet run
```

## Step 2: Authenticate

Send your first message to trigger authentication with GitHub Copilot.

## Step 3: Chat with Your Agent

```
> Hello! Can you help me create a simple C# class?
```

The agent will respond with code and explanations.

## Step 4: Use Tools

```
> Can you read the README.md file and summarize it?
```

The agent can use tools like `read`, `write`, `bash`, `grep`, etc.
