# Getting Started with Pi-Mono-Net

Welcome to Pi-Mono-Net! This guide will help you get up and running with your AI coding agent.

## Prerequisites

- **.NET 8.0 SDK** or later
- **Git** for cloning the repository  
- **GitHub Copilot subscription** (recommended) or OpenAI API key
- **Visual Studio Code** or your preferred IDE (optional)

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/ar4ft/pi-mono-net.git
cd pi-mono-net
```

### 2. Build the Solution

```bash
dotnet build
```

### 3. Run the Coding Agent

```bash
cd src/Pi.CodingAgent
dotnet run
```

## First Run

On first run, you'll be prompted to authenticate with GitHub Copilot.
