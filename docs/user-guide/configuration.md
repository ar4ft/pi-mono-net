# Configuration Guide
Settings, preferences, and customization for Pi-Mono-Net.

## Configuration Files
- `~/.pi/settings.json` - User preferences
- `~/.pi/auth/` - Encrypted credentials
- `~/.pi/sessions/` - Conversation history

## Default Settings
```json
{
  "defaultModel": "gpt-4",
  "defaultProvider": "github-copilot",
  "theme": "dark",
  "autoSave": true,
  "autoSaveInterval": 30,
  "maxHistorySize": 1000
}
```

See [Getting Started](getting-started.md) for more details.
