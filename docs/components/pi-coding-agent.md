# Pi.CodingAgent - Interactive CLI

Interactive CLI with REPL and tools.

## Features
- REPL loop
- Command system
- Session management
- Agent tools (bash, file ops, grep, etc.)
- Settings management
- Auth storage

## Usage
```bash
cd src/Pi.CodingAgent
dotnet run
```

## Commands
- `/help` - Show help
- `/models` - List models
- `/history` - Show history
- `/save` - Save session
- `/load` - Load session

## Agent Tools
- `bash` - Execute commands
- `read` - Read files
- `write` - Write files
- `edit` - Edit files
- `grep` - Search content
- `find` - Find files
- `ls` - List directory
